# Методология аудита

## Основана на стандартах

- **МСА 200** — Общие цели аудитора
- **МСА 240** — Ответственность в отношении мошенничества
- **CISA Standard 1210** — Профессиональная компетентность
- **Положение ЦБ 716-П** — Качество данных

## Процесс

1. Сбор доказательств (RAG + Web Search)
2. Анализ 100% транзакций (Python/Pandas)
3. Оценка рисков (5x5 матрица)
4. Генерация отчета (Map-Reduce)
